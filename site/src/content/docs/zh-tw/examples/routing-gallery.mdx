## 提交案例

如果你想提議新的 public-safe 範例，請使用 route case issue template。請不要放入真實客戶名稱、本機路徑、主機名稱、憑證或私人 repository 細節。

開 PR 前先跑：

```bash
python scripts/validate-route-cases.py route-cases
python scripts/build-route-gallery.py --check
```
