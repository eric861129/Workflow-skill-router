title: Sample Skills
description: 對應常見工程 routes 的可複製公開 SKILL.md 範例。
---

Repo 內包含幾個小型、generic 的 sample skills，讓讀者可以看到被 router 選中的 skill 實際可以長什麼樣子。

```text
sample-skills/
  api-guidelines-skill/
  vue-composition-patterns-skill/
  frontend-debugging-workflow/
  docker-compose-local-dev-skill/
  requirements-clarity/
  systematic-debugging/
```

這些範例刻意保持通用，不包含組織名稱、私有路徑、hostnames、secrets 或特定產品架構。

你可以把它們當成建立自己 skill catalog 時的 reference implementations。
