## Submit a case

Use the route case issue template when you want to propose a new public-safe example. Keep real customer names, local paths, hostnames, credentials, and private repository details out of the submission.

Run these checks before opening a pull request:

```bash
python scripts/validate-route-cases.py route-cases
python scripts/build-route-gallery.py --check
```
