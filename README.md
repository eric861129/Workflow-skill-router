## What This Project Does

Workflow Skill Router is a method for organizing many AI Agent skills into a small, explainable routing layer.

Core model:

```text
Task nature
  -> Work stage
    -> Technical domain
      -> 1-4 actual skills
```

It helps an AI Agent decide which skills to use before starting work, without turning every task into a noisy multi-skill workflow.
