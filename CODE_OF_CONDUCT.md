# Code Of Conduct

## Our Pledge

We want Workflow Skill Router to be useful to developers who build, share, and maintain AI agent workflows. We welcome people with different tools, platforms, languages, and levels of experience.

## Expected Behavior

- Be respectful and specific.
- Critique ideas, examples, and routing rules, not people.
- Share reproducible prompts, routes, logs, or screenshots when reporting a routing failure.
- Keep private organization names, credentials, paths, and customer data out of public issues and pull requests.

## Unacceptable Behavior

- Harassment, insults, threats, or personal attacks.
- Publishing private or sensitive information without permission.
- Spam, unrelated promotion, or repeated low-effort comments.
- Deliberately misleading security or privacy reports.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, discussions, and pull requests that violate this code of conduct. Repeated or severe violations may result in a ban from the project community.

## Reporting

If you see behavior that violates this code of conduct, contact the maintainer through GitHub. For sensitive security or privacy issues, follow `SECURITY.md` instead of posting details publicly.
