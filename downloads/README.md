# Download Packages

These archives are generated by `python scripts/package-downloads.py`.

## Packages

- `workflow-skill-router-blank.zip`: a ready-to-install blank `workflow-skill-router/` starter skill.
- `workflow-skill-router-template.zip`: the blank starter plus the common engineering router and public sample skills.

The template package is built from the public-safe material already committed in this repository. It intentionally excludes private organization-specific skills, paths, deployment details, and internal identifiers.

## Regenerate

```bash
python scripts/package-downloads.py
```
