# Download Packages

These archives are generated by `python scripts/package-downloads.py`.

## Packages

- `workflow-skill-router-blank.zip`: a ready-to-install blank `workflow-skill-router/` starter skill.
- `workflow-skill-router-template.zip`: a public-safe export of the maintainer's real local Codex skills catalog.
- `workflow-skill-router-template-manifest.md`: included skill folders, excluded private skill count, and sanitization summary.

The template package includes `skills/`, with a sanitized `workflow-skill-router` and public-safe copies of the maintainer's real skills. It intentionally excludes private organization-specific skills and omits private lines from otherwise public skill files.

## Regenerate

```bash
python scripts/package-downloads.py --skills-root <path-to-local-codex-skills> --exclude-prefix <private-prefix> --exclude-name <private-skill-name> --private-marker <private-text-marker>
```
