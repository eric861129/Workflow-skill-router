# Company Platform Sanitized Router

An anonymized company platform router that preserves real-world complexity without exposing private project names, paths, or deployment details.

## Audience

Use this if your company has multiple internal apps, customer portals, backend services, documentation workflows, CI/CD rules, and live data constraints.

## Sample Prompts

- "Migrate a legacy customer workflow into the new platform API and portal."
- "Debug an RBAC mismatch between staging data and the admin UI."
- "Prepare a release of the internal operations dashboard."
