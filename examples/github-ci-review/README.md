# GitHub CI Review Router

Router for PR review comments, CI failures, branch readiness, release checks, and remote GitHub evidence.

## Audience

Use this for teams that need agents to work with PRs, checks, reviews, release branches, and CI logs without confusing local code review with remote GitHub state.

## Sample Prompts

- "Address the unresolved PR review comments."
- "Fix the failing GitHub Actions build."
- "Check whether this release branch is ready to merge."
