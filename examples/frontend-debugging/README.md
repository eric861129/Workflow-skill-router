# Frontend Debugging Router

Router for choosing between UI debugging, interactive browser inspection, scripted automation, framework expertise, and backend/API investigation.

## Audience

Use this for web app teams debugging customer portals, internal admin tools, dashboards, or single-page applications.

## Sample Prompts

- "The checkout button disappears only after filtering the grid."
- "The admin page works locally but fails after login in Chrome."
- "Create a repeatable regression check for this UI bug."
