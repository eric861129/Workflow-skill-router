# Codex Workflow Skill Router

Codex-specific example router for users who want to wire the pattern into a Codex skill folder.

## Audience

Use this when your agent environment is Codex and your available skills include local custom skills, plugin skills, connector skills, and meta workflow skills.

## Sample Prompts

- "Use the right skills before implementing a backend API change."
- "Debug this local frontend issue and verify it in the browser."
- "Address GitHub PR review comments and keep the commit clean."
