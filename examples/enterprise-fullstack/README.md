# Enterprise Fullstack Router

Company-focused router for product teams that maintain backend services, frontend apps, documentation, CI, and release workflows.

## Audience

Use this for SaaS, internal platforms, enterprise portals, operations dashboards, and revenue systems.

## Sample Prompts

- "Add a customer export API and update the portal client."
- "Plan a release for the billing dashboard."
- "Write technical docs for a new integration workflow."
