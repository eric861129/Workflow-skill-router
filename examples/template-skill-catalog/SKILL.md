# Template Skill Catalog Router

這是 legacy/tutorial，不是 runtime discovery。從 `capability-catalog.example.json` 選一個 Primary 與最多三個 Supporting；額外能力仍需使用者同意。

小型 API 規劃可用 `api-designer`；多階段工作以 `requirements-clarity` 規劃後，在驗證階段改用 `qa-test-planner`。瀏覽器回歸使用 `playwright`，問題定位使用 `systematic-debugging`。
