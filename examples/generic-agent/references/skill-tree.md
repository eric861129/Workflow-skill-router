# Workflow Skill Tree

## Requirements / Planning

- Requirements / Clarify / Small feature: Primary: `requirements-clarity`; Supporting: `implementation-planning`
- Planning / Break down / Multi-step task: Primary: `implementation-planning`; Supporting: `test-planning`

## Engineering

- Backend / API / Contract design: Primary: `api-designer`; Supporting: `backend-developer`, `test-planning`
- Frontend / UI / New screen: Primary: `frontend-builder`; Supporting: `ui-design`, `browser`
- Debugging / Root cause / Unknown failure: Primary: `systematic-debugging`; Supporting: `browser`, `test-runner`

## Documentation / Review

- Documentation / Technical guide / Developer audience: Primary: `documentation-writer`; Supporting: `diagramming`
- Review / Pull request / Code feedback: Primary: `code-review`; Supporting: `github-connector`

## Release

- Release / Readiness / Final checks: Primary: `release-checklist`; Supporting: `test-runner`, `git-workflow`
